软件简介

软件名称：Pindlebot SIG版
操作系统：Windows 9x/ME/2000
运行环境：SIG v1.15、Diablo II: Lord of Destruction
硬件要求：须能满足“Diablo II: Lord of Destruction”正常运行.

Pindlebot SIG版是Pindlebot 3.47的移植版本，运行于SIG 1.15环境，在速度和稳定性方面比Pindlebot 3.47有极大的提高。用户计算机上必须已经安装了SIG 1.15，否则Pindlebot SIG版无法正常运行。

Pindlebot SIG版新特性

虽然Pindlebot SIG版是Pindlebot 3.47的移植版本，但作者在移植过程中也对它作了一些修改，主要包括以下几方面：

  1. 删除了“Pickit.d2h”模块，Pindlebot SIG版内置了物品监测和拾取函数，不再使用“Pickit”来完成这些功能，这样做的好处是在杀死Pindleskin后可以实时监测到怪物是否掉了有价值的物品，如果没有发现任何有价值的物品，程序将立即退出游戏而不必再无谓地等待好几秒。
  2. 删除了“Bind.d2h”，Pindlebot SIG版内置了快捷键拦截函数来捕获F9、F10、F11、F12和Pause按键，这样可以避免这些热键被其它程序所篡改。
  3. 用户可以为每一个角色指定是否使用“消耗性武器(Arrow/Bolt)”，并指定每次购买箭矢的数量。
  4. 删除了角色(Sorceress)在Pindleskin死亡后的那一次Teleport。
  5. 加快了城中行走的速度。

程序文件

Pindlebot SIG版包括以下文件：

  • Pindlebot.d2h ---- 游戏内主控模块，负责角色动作、安全保障、物品拾取、快捷键拦截等功能。
  • Pindlebot.exe ---- 图形化设置界面，提供简洁明了、人性化的设置界面和统计界面。
  • Mover.d2h ---- herzog_zwei的作品，用以移动尺寸为1x1的物品。
  • Pickit.ini ---- 物品拾取设置文件，可自行修改。
  • Pindlebot.chm ---- 本帮助文件。

安装方法

将Pindlebot SIG版的4个文件复制到SIG安装目录中的“Hacks”文件夹里，如果存在同名文件，请先将原来存在的文件改名。

使用方法

运行Pindlebot.exe进行必要的设置，然后运行SIG即可。Pindlebot.exe将驻留在系统托盘区，用户可随时以鼠标右键激活其功能菜单。Pindlebot.exe的功能菜单包含：

  • 物品纪录 ---- 查看怪物都掉了哪些物品。
  • 运行统计 ---- 对Pindlebot SIG版的运行数据进行计算和统计。
  • 设置 ---- 激活Pindlebot SIG版设置界面。
  • 使用帮助 ---- 打开本帮助文件。
  • 关于 ---- 软件信息界面。
  • 退出 ---- 关闭Pindlebot.exe。不影响Pindlebot SIG版的运行，只是将Pindlebot SIG版图标从系统托盘区删除。

Pindlebot SIG版驻留于系统托盘区是为了方便用户随时查看物品纪录和运行统计，驻留程序对系统资源的占用微乎其微，在运行过程中您不必将它关闭。

Pindlebot SIG版的设置界面与Pindlebot 3.47基本相同，因此无需再作详细地说明。如果您对设置有问题，请参阅Pindlebot 3.47的使用文档。

游戏内快捷键

和Pindlebot 3.47一样，Pindlebot SIG版也提供了游戏内快捷键以方便用户在游戏内部对程序的运行进行常用操作：

  • F9 ---- 查看运行统计。
  • F10 ---- 最小化Diablo II窗体。
  • F12 ---- 停止Pindlebot SIG版的运行。
  • Pause ---- 暂停Pindlebot SIG版的运行。

注意：所有的游戏内快捷键都只有在角色进入游戏后才起作用。
